# CLAUDE.md — MCP Airflow

## Projeto

MCP server que expoe Apache Airflow REST API como tools. Construido com FastMCP.

## Stack

Python | FastMCP

## Status

Ativo. Ferramenta interna para operacao de DAGs.

## Setup obrigatório após clone

Executar uma vez: `git config core.hooksPath .githooks`

Isso ativa o pre-push hook que exige bump do arquivo `VERSION` quando há mudanças de código.
